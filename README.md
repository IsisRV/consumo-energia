# ⚡ Calculadora de Consumo de Energia

## 📌 Objetivo
Este projeto calcula o consumo mensal de energia elétrica de um aparelho, ajudando o usuário a estimar gastos.

## 💻 Linguagem
- Python 🐍

## 🧮 Fórmula utilizada
consumoMensal = (potencia * horasDia * 30) / 1000
